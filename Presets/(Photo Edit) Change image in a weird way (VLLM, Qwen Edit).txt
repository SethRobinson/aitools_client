#Vision LLM analyzes image then generates surreal edit prompt for Qwen

#=== EDIT THIS LINE TO CUSTOMIZE THE EDIT STYLE ===
%edit_style%="weird, surreal, or unexpected"

%max_size%="1024"

#Resize source image if too large for vision LLM
command @resize_if_larger|image1|x|%max_size%|y|%max_size%|aspect_correct|1|

#Step 1: Vision LLM describes the image
command @llm_prompt_reset
command @llm_prompt_set_base_prompt
You are an expert image analysis assistant with exceptional attention to visual detail. Your task is to provide comprehensive, richly detailed descriptions that capture every significant element of an image. Focus on creating descriptions that would allow someone to recreate or edit the image accurately.
@end
command @llm_add_image|image|
command @llm_prompt_add_from_user
Please provide a thorough and detailed description of this image. Include the following elements:

SUBJECT: Identify all people, animals, or main objects. For people, describe their apparent age, gender, ethnicity, body type, hairstyle and color, facial features, expression, pose, and body language. Note what they are doing and any interactions between subjects.

CLOTHING & ACCESSORIES: Describe all clothing items including style, color, texture, fit, and condition. Note jewelry, glasses, hats, bags, or other accessories.

SETTING & ENVIRONMENT: Describe the location (indoor/outdoor, specific type of place), time of day, weather conditions if visible, architectural elements, furniture, vegetation, or background details.

COMPOSITION & PHOTOGRAPHY: Note the framing, camera angle, depth of field, lighting style (natural, studio, dramatic, soft), and any notable photographic techniques.

COLORS & MOOD: Describe the dominant color palette, color relationships, overall atmosphere, and emotional tone of the image.

NOTABLE DETAILS: Mention any text, logos, unusual elements, artistic choices, or distinctive features that stand out.

Write your description as flowing prose, not a list.
@end
call_llm

%image_description%=%llm_reply%

#Step 2: LLM creates edit prompt based on style
command @llm_prompt_reset
command @llm_prompt_set_base_prompt
You are a creative image editor. Given an image description, you will suggest ONE specific, detailed edit that transforms the image.

The user wants the edit to be: %edit_style%

Your edit should match that style while being visually striking and imaginative.

RULES:
- Output ONLY the editing instruction itself, nothing else
- Be specific about colors, materials, sizes, and placement
- Make it detailed enough that an AI image editor can execute it precisely
- Aim for 200-400 characters of text in your reply
- Don't explain why or add commentary
- Stay true to the requested style
@end
command @llm_prompt_add_from_user|%image_description%|
call_llm

#Step 3: Apply the edit
%prompt%=%llm_reply%
img_to_img_qwen_edit.json @upload|image1|input1|
